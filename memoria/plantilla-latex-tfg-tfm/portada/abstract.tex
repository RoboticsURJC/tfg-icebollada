\chapter*{Abstract\markboth{Abstract}{Abstract}}
Thanks to technological development, robotics is becoming increasingly important in different fields, simplifying people's work due to the efficiency and automation of systems previously managed by humans.\\

One of the key fields in robotics is artificial intelligence, which aims to replicate the behavior of the human brain. At the same time, one of its most important fields is artificial vision, which aims to imitate the behavior of human vision. For this, the training of algorithms is important. The most used are Deep Learning algorithms, which try to replicate the structure of neurons.\\

For this project, a monitoring system has been created to control and study the welfare of mice. Researchers from the UAH Animal Welfare and Research Laboratory have been contacted to learn about the real requirements, which have been analyzed and solved by developing a user interface that offers the data collected by the system in a simple way.\\

For the implementation of this system, the Raspberry Pi 4 has been used, which allows the system to be low-cost. The interface has been created in Node-Red, where the readings of the different sensors have been collected as well as the web server of the cameras, created with Flask. In addition, a dataset with images of mice has been created, which has allowed training a detection model in YOLO, trained through a convolutional neural network, a type of Deep Learning algorithm.\\

The result has been a user friendly interface. Despite this, there are improvements that could be applied in the future to the developed system, such as the use of a Docker for a quick and easy installation by any user, the analysis of animal behavior with Deep Learning, or access to the interface user from anywhere.
